Tutorial Pengolahan Data

  1. Data penyakit menular bersumber dari https://opendata.jabarprov.go.id/
  2. Ambil dataset penyakit menular ,yaitu Malaria, Tuberkulosis, dan HIV pada tahun 2021
    1. Malaria
      https://opendata.jabarprov.go.id/id/dataset/jumlah-kasus-penyakit-malaria-berdasarkan-kabupatenkota-di-jawa-barat
    2. Tuberkulosis
      https://opendata.jabarprov.go.id/id/dataset/jumlah-kasus-penyakit-tuberkulosis-berdasarkan-kabupatenkota-di-jawa-barat
    3. HIV
      https://opendata.jabarprov.go.id/id/dataset/jumlah-kasus-hiv-berdasarkan-kabupatenkota-di-jawa-barat
  3. Lalu, terdapat data pendukung sebagai gambaran jumlah tenaga kesehatan dan fasilitas pelayanan kesehatan pada tahun 2021
    1. Tenaga Kesehatan
      https://subangkab.bps.go.id/statictable/2022/09/12/292/jumlah-tenaga-kesehatan-menurut-kabupaten-kota-di-provinsi-jawa-barat-2021-.html
    2. Fasilitas Pelayanan Kesehatan
      https://opendata.jabarprov.go.id/id/dataset/jumlah-fasilitas-kesehatan-berdasarkan-jenis-di-jawa-barat
  4. Download dataset yang diperlukan dalam bentuk xls. dan lakukan proses data cleaning
    Cleaning adalah Proses cleaning data atau pembersihan dilakukan terhadap data yang redudan atau ganda, inkonsisten, missing value dan outlier data.
  5. Setelah data yang dibutuhkan sudah tersedia, tahap selanjutnya proses visualisasi data

Visualisasi data dilakukan melalui tools https://lookerstudio.google.com/

  1. Membuat lembar kerja baru (blank report).
  2. Ganti nama lembar Project Day 1.
  3. Pilih New Page untuk menampilkan lembar lebih dari 1.
  4. Klik Add Data.
  5. Lalu, pilih add data yaitu menggunakan format file dari Google Sheet.
  6. Pilih file sesuai dengan data yang diambil.
  7. Pilih worksheet data yang akan diolah pada spreadsheet.
  8. Tambahkan worksheet yang dipilih.
  9. Pada bagian kanan tampilan terdapat menu DATA, klik worksheet yang telah ditambahkan.
  10. Lakukan edit data untuk mengubah tipe yang dibutuhkan (untuk mempermudah visualisasi).
  11. Pilih menu Add a chart pada bagian atas lembar kerja dan pilih Pivot Table.

A. HALAMAN UTAMA

Halaman Utama pada website menggunakan visualisasi data dalam bentuk grafik

  1. Setelah Table tersedia, pada item chart Row Dimention tambahkan atribut “id”, “nama_kabupaten_kota”.
  2. Pada item chart Metric tambahkan “jumlah_kasus_malaria”, “jumlah_kasus_TBC”, “Jumlah_kasus_HIV”, “jumlah_dokter”, “jumlah_RSU”, “jumlah_Puskesmas”.
  3. Salin link pada embed report dan masukkan pada source code html.

B. HALAMAN KASUS

Pada halaman kasus menggunakan visualisasi data dalam bentuk peta

  1. Setelah Table tersedia, pada item chart Row Dimention tambahkan atribut “nama_kabupaten_kota”.
  2. Pada item chart Metric tambahkan “jumlah_kasus_malaria” atau “jumlah_kasus_TBC” atau “Jumlah_kasus_HIV”.
  3. Salin link pada embed report dan masukkan pada source code html.

Website laporan penyakit menular Provinsi Jawa Barat akan berisi tampilan peta dan grafik yang memuat persebaran penyakit, tenaga kesehatan, dan fasilitas pelayanan kesehatan berdasarkan kabupaten/kota Provinsi Jawa Barat. Pembuatan website memiliki
  1. Menggunakan bahasa pemrograman server, PHP (Hypertext Preprocessor).
  2. Menggunakan bahasa pemrograman client, HTML (Hypertext Markup Language) dan CSS(Cascading Stylesheet).
  3. Tools yang digunakan xampp, vscode, web browser google chrome.
  4. Referensi code (Youtube, Bootstrap, dan W3schools)
